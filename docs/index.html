AppCanvas

Implements the ICanvas interface to provide drawing operations on a canvas. This class supports operations such as drawing shapes (circle, rectangle), setting pen properties (color, size), and interacting with a bitmap canvas.

AppCommandFactory

Factory class that creates command objects, extending the base BOOSE.CommandFactory to support custom command types, such as the "write" command.

AppIf

Represents a compound command for handling 'If' statements. This class manages the creation of 'If' instances and checks for limits on the number of instances.

Command class that implements text writing functionality on a canvas. Inherits from BOOSE.CommandOneParameter to handle single parameter text commands.

Form1

The main form of the drawing application. It handles user interactions, processes commands, and renders the drawing on a canvas.

MyArray

Represents a custom array class that extends the BOOSE.Array class. This class restricts the number of instances that can be created to a maximum of 100. It includes methods to manage and enforce these restrictions.

MyElse

Represents a custom implementation of the BOOSE.Else class. This class overrides the Restrictions() method and includes functionality to manage restrictions by calling ReduceRestrictions.

MyEnd

Represents a custom implementation of the BOOSE.End class. This class overrides the Restrictions() method and includes functionality to manage restrictions by calling ReduceRestrictions.

MyFor

MyInt

Represents a custom implementation of the BOOSE.Int class. This class overrides the Restrictions() method and enforces a limit on the number of instances based on the static ASE_Assignment.MyInt.myintcount variable.

MyMethod

Represents a custom implementation of the BOOSE.Method class. This class overrides the Restrictions() method and enforces a limit on the number of instances based on the static MethodCount variable.

MyReal

Represents a custom implementation of the BOOSE.Real class. This class overrides the Restrictions() method and enforces a limit on the number of instances based on the static ASE_Assignment.MyReal.myRealCount variable.

MyStoredProgram

Represents a custom implementation of the BOOSE.StoredProgram class. This class overrides the Run() method and provides logic for executing commands while enforcing execution limits. It also overrides the Push(ConditionalCommand) and Pop() methods to manage the stack of conditional commands.

MyTriangle

Represents a command to draw a triangle on the canvas. This class overrides the BOOSE.CommandThreeParameters and implements the Execute() method to draw a triangle given its X and Y coordinates and side length.
